Used this Flask Tutorial to setup the Flask server
https://towardsdatascience.com/creating-restful-apis-using-flask-and-python-655bad51b24

1. Install Flask "pip3 install Flask"
2. Write app.py file
3. Run server: "python3 app.py"
4. Command to run: FLASK_APP=app.py FLASK_ENV=development flask run


Used to deploy to heroku:
https://dev.to/lordofdexterity/deploying-flask-app-on-heroku-using-github-50nh

Additional libraries:
1.  Install "pip3 install pandas"
2.  Install "pip3 install requests"
3.  Install "pip3 install flask_wtf"
4.  Install "pip3 install wtforms"
5.  Install "pip3 install geopy"
6.  Install "pip3 install flask_cors"
